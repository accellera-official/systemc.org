## 🚀 What is the SystemC Sprint?

The Accellera SystemC Sprint is a practical, code‑focused event where participants actively develop and advance key SystemC initiatives. Unlike a conference or workshop, this sprint is designed for collaborative implementation, discussion, and rapid prototyping. Participants are expected to write, review, and discuss real SystemC code together with fellow contributors and experts.

## 📅 Event Overview
- Format: Online event
- Date and Time: 22 April 2026, 14:00 - 17:00
- Audience: SystemC developers, contributors, and technical users
- **NOTE**: Registration for this event has been closed

## 🎯 Sprint Goals
- Advance concrete SystemC technical deliverables
- Enable direct collaboration between contributors
- Kick‑off new initiatives with shared technical direction
- Produce working proof‑of‑concept code
